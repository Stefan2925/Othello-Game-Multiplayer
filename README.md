How to play:
Pentru a ajunge in punctul in care tabla cu piese este generata se poate face prin mai multe moduri .Primul mod este prin apasarea butonului „Play Offline” , iar a doua metoda este prin metoda Multiplayer(Retea) adica in cazul in care vrei sa dai Host la Server, vei apasa pe „Play Online” si odata ce fereastra de retea e generata se apasa pe „Host” si se asteapta conectarea clientului.In cazul in care esti client se apasa la fel pe „Play Online” si se introduce adresa IP a host-ului , nu e necesar si port-ul, deoarece este determinat automat de catre program, apoi se apasa pe „Connect”.

In cazul in care nu sunt doua calculatoare, se poate rula de doua ori programul, simuland "multiplayer".
